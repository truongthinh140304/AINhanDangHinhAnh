━━━ Cách 1: Nhấn F5 (Khuyên dùng ⭐) ━━━

   3. Chọn: "🐍 Start Backend Server"


╔════════════════════════════════════════════════════════════════╗
║             📱 CHẠY FLUTTER APP (TRÊN ĐIỆN THOẠI)             ║
╚════════════════════════════════════════════════════════════════╝

━━━ BƯỚC 1: Kết Nối Điện Thoại ━━━

📱 BẬT DEVELOPER MODE:

   Samsung:
      Settings → About Phone → Software Information 
      → Nhấn "Build Number" 7 lần

   Xiaomi/Redmi:
      Settings → About Phone 
      → Nhấn "MIUI Version" 7 lần

   Oppo/Realme:
      Settings → About Phone 
      → Nhấn "Version" 7 lần

   Khác:
      Settings → About Phone 
      → Nhấn "Build Number" 7 lần


📱 BẬT USB DEBUGGING:

   Settings → Developer Options
   → Bật "USB Debugging"
   → Bật "Install via USB" (nếu có)


📱 CẮM USB:

   1. Cắm USB vào máy tính
   2. Điện thoại hỏi "Allow USB debugging?"
      → Chọn "Always allow from this computer"
      → OK
   3. Chọn USB mode: "File Transfer" hoặc "MTP"


📱 KIỂM TRA:

   # Trong VS Code terminal (Ctrl + `)
   flutter devices

   ✅ Sẽ thấy điện thoại trong danh sách:
      SM G973F (mobile) • XXXXXX • android-arm64


━━━ BƯỚC 2: Chạy App (3 Cách) ━━━

━━━ Cách 1: Nhấn F5 (Khuyên dùng ⭐) ━━━

   1. Nhấn F5
   2. Chọn: "📱 Flutter App"
   3. ✅ App sẽ cài và chạy trên điện thoại!


━━━ Cách 2: Dùng Terminal ━━━

   cd frontend
   flutter run

   ✅ Flutter sẽ tự detect điện thoại và chạy!


━━━ Cách 3: Mở main.dart + F5 ━━━

   1. Mở file: frontend/lib/main.dart
   2. Nhấn F5
   3. ✅ Flutter tự chạy!

╔════════════════════════════════════════════════════════════════╗
║                   🔧 TROUBLESHOOTING                          ║
╚════════════════════════════════════════════════════════════════╝

❌ "Python interpreter not found"
   → cd backend
   → python -m venv venv
   → Ctrl+Shift+P → "Python: Select Interpreter"
   → Chọn: ./backend/venv/Scripts/python.exe

❌ "No device found" 
   → Kiểm tra USB debugging đã bật
   → Chạy: adb devices
   → Nếu không thấy: adb kill-server && adb start-server

❌ "Module not found" (Python)
   → cd backend
   → pip install -r requirements.txt

❌ Flutter packages error
   → cd frontend
   → flutter clean
   → flutter pub get

❌ Port 8000 đã dùng
   → Đổi port trong backend/main.py:
      uvicorn.run(app, host="0.0.0.0", port=8001)
